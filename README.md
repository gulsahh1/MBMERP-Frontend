# MBM ERP Frontend

MBM ERP sisteminin frontend uygulamasıdır.  
Modern web teknolojileri ile geliştirilmiş, backend API ve real-time SignalR entegrasyonu olan yönetim panelidir.

---

## 🚀 Teknolojiler

- React (Vite)
- TypeScript
- React Router DOM
- Axios
- Context API / State Management
- Tailwind CSS
- SignalR Client
- JWT Authentication
- Vite

## ⚙️ Özellikler

- JWT tabanlı login sistemi
- Role-based erişim kontrolü
- Dashboard yönetimi
- CRUD operasyonları
- Backend API entegrasyonu
